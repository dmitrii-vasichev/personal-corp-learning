# Связи Issue ↔ PR ↔ Коммиты в GitHub

## Упоминания (#N)

Напиши `#N` в любом месте (коммит, PR, Issue, комментарий) — GitHub создаст перекрёстную ссылку.
Issue НЕ закрывается, только связывается.

## Closing Keywords (автозакрытие)

| Keyword | Варианты |
|---------|----------|
| Close   | Close #N, Closes #N, Closed #N |
| Fix     | Fix #N, Fixes #N, Fixed #N |
| Resolve | Resolve #N, Resolves #N, Resolved #N |

**Правила:**
- Работает в теле PR (не в заголовке)
- Работает в коммит-сообщениях при merge в default branch
- Можно закрыть несколько: `Closes #1, closes #2`
- Регистр не важен

## Связь через имя ветки

Конвенция: `тип/номер-описание`
- `feature/15-add-auth`
- `fix/16-broIssue #15 закрыт


Из любой точки можно пройти всю цепочку в обе стороны.
